# 🌐 Language Translator using Python

A simple Language Translator application built with Python. This project allows users to translate text from one language to another using a translation library.

## Features
- Translate text into multiple languages
- Easy to use
- Fast translation
- Simple Python implementation

## Technologies Used
- Python
- Deep Translator (or Googletrans)
